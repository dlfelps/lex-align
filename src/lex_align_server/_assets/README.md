# lex-align server (operator bundle)

This directory was generated by `lex-align-server init`. It contains
everything needed to bring up a single-user `lex-align` server.

## Bring it up

```bash
cp .env.example .env          # edit if you need to change defaults
docker compose up -d
curl http://127.0.0.1:8765/api/v1/health
```

`lex-align-server selftest --url http://127.0.0.1:8765` verifies the
stack is alive end-to-end.

## Files

| File | Purpose |
|---|---|
| `Dockerfile` | Pulls `lex-align[server]` from PyPI. |
| `docker-compose.yml` | Server + Redis stack. Single-user mode by default. |
| `.env.example` | Copy to `.env`. Holds `AUTH_ENABLED`, `CVE_THRESHOLD`, etc. |
| `registry.yml` | Human-authored enterprise registry. Edit this. |
| `registry.json` | Compiled registry consumed by the server. Regenerate with `lex-align-server registry compile registry.yml registry.json`. |

## Updating the registry

```bash
$EDITOR registry.yml
lex-align-server registry compile registry.yml registry.json
docker compose restart lexalign-server
```
